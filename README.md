## Installation

You can install the package via composer:

```bash
composer require albertoroldanq/weight-conversions
```

## Usage

```php
$weightConversion = new Albertoroldanq\WeightConversions();
echo $weightConversion->echoPhrase('Hello, Albertoroldanq!');
```
